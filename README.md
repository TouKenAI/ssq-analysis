# SSQ Analysis（双色球选号分析）

> **双色球（中国福利彩票）标准 5 步分析流程 + 最大覆盖优化引擎**
>
> **仅供娱乐参考，不构成任何购彩建议。** ⚠️

---

## 概述（Overview）

本项目将「双色球选号分析」固化为一套 **可复现、可审计、离线友好** 的标准流程，把频率学、贝叶斯、信息熵、组合覆盖设计与蒙特卡洛模拟等数学模型落到可直接运行的 Python 脚本上。

适用于：抓取近 20 期开奖数据、统计冷热号、制定选号过滤规则、生成差异化 5 组选号、用优化引擎最大化「中奖可能性」的统计维度，以及对整条流水线做独立复查。

## 作用（Purpose）

| 功能 | 说明 |
|------|------|
| 数据存档 | 抓取/导入近 20 期开奖数据，校验格式与连续性 |
| 冷热分析 | 统计红球 01–33 出现频次，按均值±0.5标准差分类热/温/冷 |
| 过滤规则 | 奇偶比 / AC值 / 三区分布 三重硬性约束回测 |
| 组合生成 | 备选蓝球 + 差异化 5 组红球（贪心选覆盖最广） |
| 成品输出 | 5 组完整号码 + 每组详细备注（冷热配比/奇偶/AC/三区/连号） |
| 优化引擎 | 蓝球分散 / 红球覆盖广度 / 热度规避 三维优化 + 蒙特卡洛验证 |
| 独立复查 | 从原始数据独立重新推导全部指标，逐项比对防偏差 |

## 机制（Mechanism）

### 标准 5 步流程（强制顺序）

```
第1步 建库 → 第2步 红球冷热 → 第3步 过滤规则 → 第4步 蓝+红组合 → 第5步 成品输出
     ↓                                                                    ↓
  ssq_raw_20.csv（单一数据源，步骤间流转）                    5组完整号码+备注
```

### 第 6 层优化引擎（在 5 步之上叠加）

1. **末等奖保底率**：蓝球分散到 5 个不同号，「蓝对」概率 = 5/16（理论最大）
2. **红球覆盖广度**：整数规划近似贪心，5 组并集覆盖 29/33（数学上限 30）
3. **人群热度规避**：避生日簇 / 等差序列 / 往期重号

### 独立复查（发布前必跑）

每次产出后运行 `ssq_review.py`，从 `ssq_raw_20.csv` **独立重新推导**全部指标——用于捕获口径/计算偏差。

## 算法（Algorithms）

| 模块 | 数学方法 | 输入 | 输出 |
|------|---------|------|------|
| 冷热分类 | 频率统计 + 均值±0.5标准差 | 20期开奖 | 热/温/冷标签 + 遗漏期数 |
| 过滤规则 | 奇偶计数 / AC值公式 / 三区枚举 | 红球6码 | 合规/不合规判定 |
| 组合选择 | C(33,6) 枚举 + 贪心覆盖 + 去重约束 | 合规池 | 5组差异化号码 |
| 覆盖优化 | 整数规划近似 + 局部搜索 | 5组候选 | 覆盖最大化的5组 |
| 验证 | 蒙特卡洛 n=200,000 | 优化方案 vs 原方案 | 置信区间 + z检验 |

## Quick Start

```bash
mkdir ssq-work && cd ssq-work
# 下载本仓库脚本到 scripts/
git clone https://github.com/TouKenAI/ssq-analysis.git

python scripts/build_db.py            # 1.建库
python scripts/ssq_red_coldhot.py     # 2.冷热
python scripts/ssq_filter_rules.py    # 3.过滤规则
python scripts/ssq_select_groups.py   # 4.备选蓝+5组
python scripts/ssq_final_groups.py    # 5.成品
python scripts/ssq_optimize_engine.py # 6.优化引擎
python scripts/ssq_review.py          # 7.独立复查（必跑）
```

所有脚本仅依赖 Python 标准库，无需安装第三方包。
